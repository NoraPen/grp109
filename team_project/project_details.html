Project Name

Beau’s Seattle Pet Rescue

Project Description

The project is aimed at helping volunteers sign up at a local charity that helps animals. The intended audience is potential volunteers who are interested in supporting animal welfare.

Project detailed description is busy Seattle animal rescue looks forward to no longer dealing with being inundated by all the local Seattle volunteers wanting to donate time to Beau's Seattle Pet Rescue. Charity wants to have a simple form consisting of labels, textboxes, dropdowns, validation and a submission button. This is really going to streamline the charity organization's workflow process and free up time for the animal's care.

Purpose

The purpose of the site is to manage the volunteer sign-up workflow efficiently. The site will provide a form to capture user details, validate user input (phone numbers, emails, required fields), and ensure a smooth sign-up process for volunteers.
